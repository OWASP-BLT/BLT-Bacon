About BACON

Revolutionize Open Source Contributions

BACON is a cutting-edge blockchain-based token system designed to incentivize engagement and contributions within open-source ecosystems. By integrating with Bitcoin and Solana blockchains, BACON introduces a transparent, secure, and gamified environment that rewards developers and contributors for their efforts.

Multi-Chain Support

Works seamlessly with both Bitcoin (via Runes protocol) and Solana blockchains

Secure & Transparent

All transactions are recorded on-chain with full transparency and security

Easy Integration

Simple APIs and SDKs to integrate into any GitHub repository

Gamified Rewards

Earn tokens for bug reports, pull requests, and code contributions

Key Features

Everything you need to reward contributors

Incentivized GitHub Issue Resolution

  • Minting and Rewarding: BACON tokens are minted as rewards when users report issues via bug logging tools
  • Proof of Authority (POA): Transparent and fair distribution of rewards
  • Developer Rewards: Earn tokens for resolving issues and contributing code

Trademark Monitoring

  • Blockchain Integration: Monitor newly registered trademarks automatically
  • Tamper-Proof Audit Trail: All monitoring activities logged immutably on-chain
  • Early Alerts: Get notified of potential conflicts immediately

Company Scoreboard

  • Blockchain-Backed Rankings: Transparent ranking system for responsiveness
  • Public Recognition: Showcase companies with the best scores
  • Tamper-Resistant: All data secured on the blockchain

Mobile Integration

  • iOS App Support: Manage tokens and track contributions on the go
  • Real-time Notifications: Stay updated on bounties and rewards
  • Seamless Experience: Full integration with the BACON ecosystem

Integration

Get started in minutes

Integrate the BACON ecosystem into your repository with simple steps

1

Install SDK

Install BACON SDK via npm or pip package manager

npm install @blt-bacon/sdk
2

Configure

Choose your blockchain and configure settings

blockchain: 'bitcoin'
3

Set Rewards

Define reward amounts for different actions

pullRequest: 50 BACON
4

Deploy

Connect GitHub and start rewarding contributors

bacon.start()
