BACON (Blockchain Assisted Contribution Network) is a Bitcoin-based token distribution service for the OWASP BLT project. This service handles the distribution of BACON tokens using the Bitcoin Runes protocol.

üöÄ Quick Start

Prerequisites

  • Python 3.11+
  • Node.js 14+
  • uv package manager
  • Cloudflare Workers account

Installation

# Install dependencies
uv pip install -r requirements.txt

# Configure environment variables
cp .env.example .env
# Edit .env with your configuration

# Deploy to Cloudflare
uv run pywrangler deploy

üîß Configuration

The service requires the following environment variables:

  • ORD_PATH - Path to ord binary
  • BITCOIN_RPC_USER_MAINNET - Bitcoin RPC username for mainnet
  • BITCOIN_RPC_PASSWORD_MAINNET - Bitcoin RPC password for mainnet
  • BITCOIN_RPC_URL_MAINNET - Bitcoin RPC URL for mainnet
  • ORD_SERVER_URL_MAINNET - Ordinal server URL for mainnet
  • WALLET_NAME_MAINNET - Wallet name for mainnet
  • WALLET_API_PASSWORD - API password for transaction authorization
